# Golden Set — 인지기능 저하 (1차 자료 직접 추출본)

> 방법론은 osteoarthritis.md와 동일: 외부 AI값 불신 · **성분마다 PubMed 신규 검색**(DB 링크 불신) · 원문에서 표준 효과·I²·출처 직접 추출 · 추정 금지 · 영선 검증 후 `확정`.
>
> **인지 조건 특이 주의:**
> - **표준 결과 = 인지 점수 변화**(MMSE / ADAS-Cog / 기억·실행기능 SMD). 통증(OA)과 달리 도메인이 여러 개 → 어느 인지 영역인지 명시.
> - **단계/인구 구분 필수:** ① 예방(건강한 노인) ② 치료(MCI 경도인지장애) ③ 치료(치매/AD). 같은 성분도 단계별로 결과가 다름 → 분리 표기.
> - DB의 인지 성분 목록은 **OA에서 복붙된 듯**(MSM·SAM-e·칼슘·셀레늄·아연 등 무관 성분 포함, 정작 은행잎·인삼·비타민B·포스파티딜세린 누락) → DB 목록 무시, 실제 근거 성분으로 신규 조사.

## 비타민 B군 (엽산/B12/B6, 호모시스테인) × 인지 · 상태: `제안(검증대기)`
**출처:** 메타 PMID 34432056 (Nutr Rev).
- MMSE 변화 **MD +0.14 [0.04, 0.23]** (6155명) — 매우 작은 개선. **장기(>12개월) MD +0.15**, **비치매 인구에서만 +0.15**(치매 환자엔 효과 없음).
- 엽산 결핍·고호모시스테인은 치매 위험↑(관찰). → **작은 효과, 조기·비치매·장기 복용 한정.** 신뢰 보통.

## 인삼 (Panax ginseng) × 인지 · 상태: `제안(검증대기)`
**출처:** 메타 PMID 39474788 (Phytother Res).
- **기억력만** 소폭 개선 SMD **0.19 [0.02,0.36]** (고용량 0.33). **전반 인지·주의·실행기능은 효과 없음**(SMD ≈ 0.06, NS).
- → 기억 도메인 한정 소효과. 신뢰 보통.

## 크레아틴 (Creatine monohydrate) × 인지 · 상태: `제안(검증대기)` — 노인 특이 효과 ★
**출처:** 메타 PMID 35984306 (Nutr Rev), PMID 39070254 (Front Nutr 2024).
| 인구 | 기억 효과 | 비고 |
|---|---|---|
| **노인 66–76세** | **SMD 0.88 [0.22, 1.55]** (I²=83%) | 뚜렷한 개선 |
| 젊은층 11–31세 | SMD 0.03 [−0.14, 0.20] | 효과 없음 |
| 전체(성인) | 기억 SMD 0.29~0.31, 주의 0.31 | **전반 인지·실행기능은 무효** |
- → **크레아틴은 노인 기억력에 의미 있는 효과**(에너지 대사 기전), 단 기억·주의 도메인 한정. 인지 영양제 중 **노인 대상 신호로는 가장 인상적.** 신뢰 보통(I² 높음, 검증 필요).

## 오메가-3 (DHA/EPA) × 인지 · 상태: `제안(검증대기)` — 단계별로 다름
**출처:** PMID 36637075(Curr Opin Lipidol), PMID 38924283(n-3 진행 지연).
| 단계 | 효과 | 출처 |
|---|---|---|
| **예방(건강 노인)** | **비일관** — 15 RCT 중 7편 효과/8편 무효 | 36637075 |
| **MCI/인지저하 진행** | **진행 지연 SMD −0.41 [−0.59,−0.23]** (CDR) | 38924283 |
| **치매/AD** | **효과 없음** | 36637075 |
- → **단계가 핵심.** 예방용으론 근거 불충분, **이미 경도 저하가 시작된 단계에선 진행 지연 신호.** AD엔 무효. 헤드라인은 단계별 표기.

## 은행잎 (Ginkgo biloba) × 인지 · 상태: `제안(검증대기)` — 단계별 + 원료(표준화)별

**⚠️ 원료 의존성:** 임상시험은 거의 전부 **표준화 추출물 EGb761(Schwabe; 플라보노이드 24%·terpene lactone 6%)** 으로 수행. **일반/비표준 은행잎 제품은 함량 편차가 크고 그 자체의 임상 근거 없음** → 같은 "은행잎"이라도 EGb761이 아니면 데이터 적용 불가. (영선 기억대로 원료사별 차이 큼.)

| 단계 | 효과 (EGb761 기준) | 출처 |
|---|---|---|
| **예방(정상·MCI 노인)** | **치매 발생 감소 효과 없음** (HR 1.12 [0.94,1.33], NS) | JAMA GEM, PMID 19017911 |
| **치료(기존 치매/인지장애)** | 인지 WMD −2.86 [−3.18,−2.54] 소폭 개선, 신경정신증상 동반 환자 한정 | 메타 PMID 25114079 |
- → **예방용 근거 없음(대규모 결정적).** 치료용 소효과는 **EGb761 한정 + 제조사 연관(펀딩 편향)** 주의. 비표준 제품엔 적용 불가.

## 커큐민 × 인지 · 상태: `제안(검증대기)` — 사람 인지엔 효과 없음
**출처:** 메타 PMID 40579315 (J Prev Alzheimers Dis).
- **동물(쥐)**: 기억 크게 개선(SMD −1.78 등) — 전임상.
- **사람: 전반 인지 효과 없음 (SMD 0.14 [−0.78, 1.07], NS).** 일부 작업기억·처리속도 하위그룹만 소폭(CI 넓음).
- → **전임상 과대광고, 사람 인지 예방효과 없음.** (OA 통증과 달리 인지엔 무효.) 신뢰 보통(효과없음).

## 아슈와간다 (Withania) × 인지 · 상태: `제안(검증대기)` — 인지가 아니라 불안/스트레스
**출처:** 메타 PMID 36017529 (Phytother Res).
- 실제 효과는 **불안 SMD −1.55(I²93.8%)·스트레스 −1.75(I²83%)** — 인지 자체 결과 아님, 이질성 극심.
- → "브레인" 마케팅은 스트레스 완화에 기댐. 인지 직접 효과 근거 부족.

## 시티콜린 (Citicoline/CDP-choline) × 인지 · 상태: `제안(검증대기)`
**출처:** 메타 PMID 36678257 (Nutrients).
- 인지 개선 pooled SMD **0.56 [0.37,0.75] ~ 1.57 [0.77,2.37]** (민감도 분석별 폭 큼 → 불안정/이질성). 주로 **혈관성·뇌졸중후 인지장애** 대상.
- → 인지 영양제 중 신호는 가장 낫지만 **추정치 불안정·인구 특이.** 신뢰 보통(과대평가 주의).

## 비타민 E × 인지 · 상태: `제안` — 효과 없음
- 확립된 근거상 인지 저하 예방/치료 **효과 없음**, 고용량은 사망률↑ 우려. → 효과 없음.

## 그 외 (근거 약함/불충분) · 상태: `제안`
- **레스베라트롤**: 단일시험 바이오마커(CSF)·탐색적, 인지 효과 불확실(MMSE 일부 둔화 보고이나 비결정적, PMID 28086917).
- **포스파티딜세린**: 옛 연구 일부 양성, 현대 RCT 근거 빈약.
- **바코파(Bacopa)**: 기억(지연회상) 소효과 가능하나 소규모.
- **프로바이오틱스**: 신호 혼재·기전 위주, 결정적 근거 없음.
- **비타민 D**: 인지 효과 "비결정적"(null 경향), 일부 메타는 비타민D 업계 펀딩(COI).
- **DB 복붙 무관 성분(MSM·SAM-e·칼슘·셀레늄·아연·마그네슘·비타민C·멜라토닌)**: 인지 근거 없음.

## 치매 치료약 (콜린에스터분해효소억제제·메만틴) — 약 · 상태: `제안(검증대기)`
**출처:** 네트워크 메타 PMID 38640313. [pubmed.ncbi.nlm.nih.gov/38640313](https://pubmed.ncbi.nlm.nih.gov/38640313/)
- **도네페질·갈란타민·리바스티그민(ChEI), 메만틴 모두 인지 개선** — ADAS-Cog **−1.23(메만틴 20mg) ~ −3.29(갈란타민 32mg)** 점(SMD ~0.2~0.4 환산).
- ⚠️ **치매(AD) 치료 단계용** — 예방·정상 인지엔 적응 아님. 효과는 modest하나 **표준 치료(근거 확립).**
- → **인지 저하 '치료' 단계에선 약(ChEI)이 영양제보다 근거 확실.** 단 예방 단계엔 약도 영양제도 결정적 효과 없음.
- (그 외 신약: 항아밀로이드 항체 lecanemab/donanemab — 초기 AD 진행 지연, 부작용·비용 이슈. 추후 추가.)

# 종합 (인지기능 저하 1차 완료) — 약 vs 영양제, 단계별

**"뇌 건강" 영양제도 관절처럼 근거 빈약. 그리고 단계(예방/MCI/치매)에 따라 결과가 다름.**

| 성분 | 예방(건강 노인) | MCI/경도저하 | 치매/AD | 한줄 |
|---|---|---|---|---|
| **크레아틴** | **기억 ★(노인 SMD 0.88)** | — | — | **노인 기억에 가장 인상적**(도메인 한정) |
| 시티콜린 | — | 개선(혈관성) | 일부 | SMD 0.56~1.57(불안정) |
| 비타민 B군 | 작게(조기·장기) | 작게 | 무효 | MMSE +0.15, 비치매만 |
| 인삼 | 기억 0.19 | — | — | 기억 도메인만 |
| 오메가-3 | 비일관 | **진행지연 −0.41** | 무효 | 단계가 핵심 |
| 은행잎(EGb761) | **무효(HR1.12)** | 무효 | 소폭(펀딩편향) | 예방X·비표준원료 적용불가 |
| 커큐민 | 무효 | — | 무효(사람) | 전임상 과대광고 |
| 아슈와간다 | (불안/스트레스) | | | 인지 아님 |
| 비타민E·레스베라트롤·PS·바코파·프로바이오틱스·비타민D·MSM류 | 없음/불충분 | | | |

**핵심:**
1. **단계가 결정적** — 같은 성분도 예방(건강)·MCI·치매에서 다름. 예: 오메가-3는 예방엔 비일관이나 **MCI 진행지연엔 신호**, B군은 비치매에만.
2. **예방 목적**(건강한 사람이 치매 무서워 미리 먹는 경우)에 확실히 듣는 보충제는 **사실상 없음** — 그나마 **크레아틴(노인 기억)** 이 가장 나은 신호.
3. **원료 의존성** — 은행잎은 표준화 EGb761만 데이터, 일반 제품 적용 불가.
4. 시장 1순위(은행잎·오메가-3·커큐민)는 예방 근거 약함 — OA와 동일한 마케팅-근거 괴리.
