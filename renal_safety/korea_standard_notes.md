# 신질환 자료 — 한국 기준 우선 원칙 + 국가별 차이 보정

> **원칙 (영선 결정, 2026-05-30):** 국가별로 기준이 다른 경우 **대한민국 식약처 허가사항 + 한국 진료지침**을 1순위로 따른다.
> 미국 FDA / 영국 Renal Drug Handbook / KDIGO는 한국 기준이 없거나 모호할 때 보조로만.
> 이 파일은 Round 1-3 cheat sheet의 국가별 차이 항목을 한국 기준으로 보정한 노트.

---

## 1. 메트포르민 ✅ 한국 = FDA 동일 (검증 완료)

**한국 식약처 허가사항 (검증, 2026-05-30 웹검색):**
| eGFR | 한국 식약처 기준 |
|---|---|
| ≥60 | 통상 |
| 45-59 | 시작 500mg 1일 1회, **1일 최대 1000mg** |
| 30-44 | **1000mg/일 이하로 제한**, 신규 투여 권장 X |
| <30 | **금기 (투여 금지)** |
- 신기능 3-6개월마다 평가.
- 조영제 CT: 검사 당일~48시간 중단.
- **2023년 대한당뇨병학회 제9판**: 메트포르민의 "2형 당뇨 1차 치료제" 규정 삭제 (참고).
- → **Round 3 수치 그대로 유효.** 한국 기준과 일치.

출처: 식약처 허가사항, 대한당뇨병학회 2023 제9판, [메디칼타임즈](https://www.medicaltimes.com/Mobile/News/NewsView.html?ID=1163477).

## 2. 니트로푸란토인 ⚠️ 국가별 cutoff 갈림 → 한국 보수 기준

| 출처 | 금기 cutoff |
|---|---|
| 전통/제품 허가 (한국 포함) | **CrCl < 60 mL/min 금기** |
| 최근 후향 연구 제안 | CrCl < 40이 더 적절할 수 있음 (근거 미미) |
| **한국 채택** | **CrCl < 60 금기** (허가사항 보수 기준 유지) |
- 한국 식약처: 간질성 신장염·자가면역성 간독성 이상반응 추가(EMA 검토 반영).
- → **Round 3 보정:** "eGFR<30 금기"가 아니라 **"CrCl<60 금기"로 상향**. 한국은 더 보수적.

출처: 식약처 허가사항 변경, [데일리팜](https://www.dailypharm.com/Users/News/NewsView.html?ID=254431), 위키 약물정보.

## 3. 국가별로 갈릴 수 있어 한국 기준 확인 필요한 항목 (영선 최종 검수)

아래는 가이드라인/허가사항이 국가별로 다를 수 있는 항목. **영선이 약학정보원/식약처에서 최종 확인** 권장:

| 항목 | 국가별 차이 가능성 | 한국 기준 확인 포인트 |
|---|---|---|
| **아세트아미노펜 최대량** | 미국 4g, 일부 유럽 3g | 한국 허가사항 1일 최대(통상 4g, 고령/간 3g 권고) |
| **세레콕시브·NSAIDs eGFR cutoff** | KDIGO 30 vs 일부 60 | 한국 신장학회 권고 |
| **가바펜틴/프레가발린 용량구간** | FDA 라벨 = CrCl 기반 | 한국 허가사항 CrCl 구간 동일한지 |
| **알로퓨리놀 + HLA-B\*5801** | 한국인 빈도 높음(12%) → **한국은 검사 권고 강함** | 대한류마티스학회 권고 |
| **콜키친 용량** | 미국 라벨 기준 | 한국 허가사항 |
| **DPP-4i 감량 구간** | 제품별/국가별 | 한국 식약처 각 제품 허가사항 |

---

## 4. 한국이 더 보수적이거나 특이한 것 (강조)

- **니트로푸란토인:** 한국 CrCl<60 금기 (미국 최근 <40 제안보다 보수) ← 한국 따름.
- **알로퓨리놀 HLA-B\*5801:** 한국인 대립유전자 빈도 높아 **중증약물이상반응(SCAR) 위험↑** → 한국은 통풍 시작 전 유전자 검사 권고가 서구보다 강함.
- **콜린알포세레이트(글리아티린):** 한국에서 임상재평가 진행 — 신질환과 별개지만 한국 특수 이슈 (factpick 인지 카테고리).

## 적용 결과 (Round 1-3 보정)

| 약물 | 기존 기재 | 한국 기준 보정 |
|---|---|---|
| 메트포르민 | <30 금기, 30-44 ≤1000mg | ✅ 유지 (한국 동일) |
| 니트로푸란토인 | eGFR<30 금기 | ⚠️ **CrCl<60 금기로 상향** (한국 보수) |
| 알로퓨리놀 | HLA-B\*5801 권고 | ✅ 한국은 검사 권고 더 강함 (강조) |
| 아세트아미노펜 | 1일 4g(보수 3g) | 한국 허가사항 확인 후 확정 (영선) |

## 영선 최종 검수 (한국 출처 대조)
- [ ] 약학정보원(health.kr) 각 약물 신기능 용량 — 위 표 대조
- [ ] 대한신장학회 / 대한당뇨병학회 / 대한류마티스학회 진료지침
- [ ] 식약처 의약품안전나라 허가사항 (메트포르민·니트로푸란토인·콜키친 우선)
- [ ] 한국은 **CrCl(Cockcroft-Gault)** 기준 허가사항 많음 — eGFR(CKD-EPI)과 환산 차이 주의

> 메모: factpick 데이터 통합(Round 4) 시 각 약물에 `kr_source`(한국 출처) 필드 + `egfr_cutoff_kr` 별도 기록. 글로벌 기준과 다르면 둘 다 보존하되 **표시는 한국 기준 우선**.
