# 신질환자 경구 처방약 — 신기능별 용량조정 (Round 3)

> **경구 약물만.** IV/주사제 제외 (일반 환자 응대 기준).
> 약사가 "당신 eGFR이면 이 약 이 용량까지" 안내할 수 있게.
> **출처:** KDIGO 2024, Renal Drug Handbook 6e (Ashley & Dunleavy), Aronoff's 5e, 제품 허가사항(FDA/EMA/식약처), 각 질환 가이드라인(ADA, KDIGO당뇨, 통풍 ACR).
> ⚠️ 처방·용량 결정은 의사 권한. 이 자료는 약사 복약지도/감사(checking) 참고용. 최종 검수 영선.

단계: G1-G2(≥60) / G3a(45-59) / G3b(30-44) / G4(15-29) / G5(<15) / HD·PD

---

## A. 당뇨

### 메트포민 (경구)
| eGFR | 권고 |
|---|---|
| ≥60 | 통상 (최대 2000-2550 mg/일) |
| 45-59 | 유지 가능, eGFR 모니터링 (3-6개월) |
| **30-44** | **감량** — 최대 1000 mg/일, 신규 시작은 권장 X |
| **<30** | **❌ 금기** (젖산산증 위험) |
| HD/PD | ❌ 금기 |
- 조영제 검사·탈수·급성질환 시 일시 중단(sick day rule).
- 출처: KDIGO 2022 Diabetes in CKD, FDA 2016 라벨 개정, 식약처.

### DPP-4 억제제 (경구) — CKD 친화적 계열
| 약물 | 조정 |
|---|---|
| **리나글립틴 (트라젠타)** | ✅ **신기능 무관 조정 불필요** (담즙 배설) — CKD 1순위 |
| 시타글립틴 (자누비아) | 100mg→ 45-30: 50mg → <30/HD: 25mg |
| 빌다글립틴 (가브스) | eGFR<50: 50mg 1일 1회 |
| 삭사글립틴 (온글라이자) | eGFR<45: 2.5mg/일 |
- HD 환자도 경구 복용 가능 (리나글립틴은 보충 불필요).
- 출처: 각 제품 허가사항, KDIGO 2022.

### SGLT2 억제제 (다파/엠파글리플로진, 경구)
| 상황 | 권고 |
|---|---|
| eGFR ≥20 | ✅ **신보호 목적 시작 가능** (다파글리플로진 DAPA-CKD, 엠파 EMPA-KIDNEY) |
| eGFR <20 | 신규 시작 X, 복용 중이면 투석 시작까지 지속 가능 |
| 혈당강하 효과 | eGFR 낮으면 혈당강하력↓ 하나 *신장·심장 보호 효과는 유지* |
- 비뇨생식기 감염·탈수 주의. 출처: KDIGO 2024, DAPA-CKD/EMPA-KIDNEY.

### 설포닐우레아 (경구) — 주의 계열
- **글리메피리드/글리벤클라미드:** eGFR<30 저혈당 위험 ↑ → 회피 또는 최소용량.
- **글리클라지드:** 상대적 안전하나 감량.
- 출처: Renal Drug Handbook 6e.

## B. 통증 (경구) — 관절통 카테고리와 연결

### 경구 NSAIDs (이부프로펜·나프록센·세레콕시브 등)
- eGFR<30 회피, 30-60 단기·저용량만 (Round 1 상세). 만성신질환에서 계열 전체 주의.

### 아세트아미노펜
- 경구 1순위 진통제. eGFR<30은 간격 6시간↑, 1일 3g↓ (Round 2 상세).

### 트라마돌 (경구)
| eGFR | 조정 |
|---|---|
| ≥30 | 통상 (속효성 50-100mg q6h, 1일 400mg↓) |
| **<30** | **감량 + 간격 연장** — 50-100mg q12h, 1일 200mg↓ |
| HD/PD | 최대 50mg q12h, 활성대사체 축적·경련 주의 |
- 서방형은 eGFR<30 회피(축적). 세로토닌 증후군·경련 역치↓ 주의.
- 출처: Renal Drug Handbook 6e, Aronoff's 5e.

### 코데인 (경구)
| eGFR | 조정 |
|---|---|
| ≥30 | 통상이나 보수적 |
| **<30** | **감량(50%)·간격 연장 또는 회피** — 활성대사체(모르핀-6-글루쿠로니드) 축적 → 호흡억제·진정 |
| HD/PD | 회피 권장 |
- 출처: Renal Drug Handbook 6e (opioids in renal failure).

### 가바펜틴 / 프레가발린 (경구, 신경병증통)
**신배설 100% → 신기능별 정밀 감량 필수. 안 하면 진정·어지럼·낙상.**

가바펜틴:
| CrCl | 1일 용량 |
|---|---|
| ≥60 | 900-3600 mg (분3) |
| 30-59 | 400-1400 mg |
| 15-29 | 200-700 mg |
| <15 | 100-300 mg |
| HD | 100-300mg 격일 + **투석 후 보충량(125-350mg)** |

프레가발린:
| CrCl | 1일 용량 |
|---|---|
| ≥60 | 150-600 mg (분2-3) |
| 30-59 | 75-300 mg |
| 15-29 | 25-150 mg |
| <15 | 25-75 mg (1일 1회) |
| HD | 기본 + 투석 후 보충 |
- 출처: 제품 허가사항(Neurontin/Lyrica), Aronoff's 5e.

## C. 통풍

### 알로퓨리놀 (경구)
| eGFR | 권고 |
|---|---|
| ≥60 | 통상 (시작 100mg, 목표 요산 따라 증량, 최대 800mg) |
| 30-59 | 시작 50-100mg, 천천히 증량, eGFR 고려 |
| **<30** | **시작 50mg**, 매우 천천히 증량, 과민반응증후군(AHS) 위험↑ |
| HD | 50-100mg, 투석 후 보충 고려 |
- 핵심: 옛 "CrCl 기반 상한 고정"은 폐기 → **저용량 시작 후 요산 목표(<6mg/dL) 보고 증량**이 현재 권고(ACR 2020). HLA-B*5801(한국인 빈도↑) 검사 권고.
- 출처: ACR 2020 Gout Guideline, KDIGO.

### 페북소스타트 (경구)
- eGFR 경-중등도 감소 시 조정 거의 불필요(간 대사) → 알로퓨리놀 부적합 시 대안. 심혈관 고위험군 주의(CARES). 출처: 허가사항.

### 콜키친 (경구, 통풍 발작/예방)
| eGFR | 조정 |
|---|---|
| ≥30 | 발작: 1mg 후 0.5mg, 예방 0.5-1mg/일 |
| **<30** | **감량·간격 연장**, 예방 시 0.5mg 격일, 신경근병증 모니터 |
| HD | 회피 또는 최소·드물게 (제거 안 됨, 독성 축적) |
- CYP3A4/P-gp 억제제(클래리스로마이신 등) 병용 시 치명적 독성 — 약사 감사 핵심.
- 출처: Renal Drug Handbook 6e, ACR 2020.

## D. 위장 (경구) — 매우 흔한 일반약/처방

### H2 차단제 (경구)
| 약물 | 조정 |
|---|---|
| 파모티딘 | eGFR<50: 50% 감량 또는 간격 연장 (정신혼란 위험) |
| (라니티딘) | 시장 철수(NDMA) — 참고만 |
- 출처: Renal Drug Handbook 6e.

### PPI (오메프라졸·에소메프라졸·란소프라졸 등, 경구)
- 신기능별 용량조정 **불필요** (간 대사).
- ⚠️ 단 장기 사용 ↔ 급성 간질성신염(AIN)·CKD 진행 연관 관찰연구 → **장기 복용은 필요성 재평가** 톤.
- 출처: 허가사항, Lazarus 2016(관찰연구).

## E. 심혈관 (경구)

### ACEi / ARB (라미프릴·로사르탄 등, 경구)
| 상황 | 권고 |
|---|---|
| CKD + 단백뇨 | ✅ **신보호 1차 약제** (오히려 권장) |
| 시작/증량 시 | Cr 30%까지 상승은 허용, K·Cr 1-2주 후 체크 |
| eGFR<30 | 지속 가능하나 면밀 모니터링, 고칼륨혈증 주의 |
| 급성 악화·고K·탈수 | 일시 중단 |
- "신장 나쁘면 무조건 끊는다"는 오해 — 단백뇨성 CKD엔 핵심 보호약. 출처: KDIGO 2024.

### 디곡신 (경구)
| eGFR | 조정 |
|---|---|
| ≥50 | 통상 0.125-0.25mg/일 |
| **<50** | **감량** 0.0625-0.125mg/일 또는 격일, 농도 모니터(좁은 치료역) |
| HD | 저용량, 혈중농도·전해질(특히 K) 모니터 |
- 출처: Renal Drug Handbook 6e.

### 스타틴 (경구)
| 약물 | 조정 |
|---|---|
| 아토르바스타틴 | ✅ 신기능 조정 불필요 (담즙 배설) — CKD 선호 |
| 로수바스타틴 | eGFR<30: 시작 5mg, 최대 10mg/일 |
| 심바스타틴 등 | 고용량 시 근병증 주의 |
- 출처: KDIGO Lipid, 허가사항.

## F. 항생제 (경구, 외래 흔한 것만)

### 신기능 감량 필요 (경구)
| 약물 | 조정 |
|---|---|
| **아목시실린 / 아목시실린-클라불란산** | eGFR<30 간격 연장(q12h), <10 더 감량 |
| **세파클러·세팔렉신 (경구 세파)** | eGFR<30-50 감량/간격 연장 |
| **시프로플록사신 (경구 퀴놀론)** | eGFR<30: 50% 또는 q24h |
| **레보플록사신** | eGFR<50 감량(첫 용량 후 절반) |
| **TMP-SMX (박트림)** | eGFR 15-30 50% 감량, <15 회피, 고K 주의 |
| **니트로푸란토인** | **eGFR<30 ❌ 금기** (효과↓·독성↑) |
| **발라시클로버/아시클로버(경구)** | 신경독성(혼란·환각) 위험 → 감량 필수 |

### 조정 불필요 (경구)
- 아지스로마이신, 클래리스로마이신(단 약물상호작용 주의), 독시사이클린, 클린다마이신, 메트로니다졸(중증 신부전서만 감량).
- 출처: Sanford Guide, Renal Drug Handbook 6e, 허가사항.

## 핵심 인사이트 (환자 안심 메시지)

**"신장 나빠도 비교적 안심하고 쓸 수 있는 경구약" (조정 적음):**
- 당뇨: **리나글립틴**(조정 불필요), SGLT2i(eGFR20까지 신보호)
- 진통: **아세트아미노펜**
- 콜레스테롤: **아토르바스타틴**(조정 불필요)
- 위장: **PPI**(용량조정 불필요)
- 신보호: **ACEi/ARB**(단백뇨엔 오히려 권장)

**"용량만 맞추면 쓸 수 있는 것" (감량으로 해결):**
- 가바펜틴·프레가발린, 트라마돌, 알로퓨리놀, 디곡신, 시타글립틴, 대부분 경구 항생제

**"신장 나쁘면 피해야 하는 것":**
- 메트포민(<30), NSAIDs(<30), 니트로푸란토인(<30), 코데인(<30 회피), 설포닐우레아 일부

→ factpick 메시지: **"신질환자도 '용량을 맞추면' 쓸 수 있는 약이 대부분. 무조건 못 먹는 게 아니다."**

## ⚠️ 데이터 정직성 (SOP 준수)
- 이 표의 처방약 수치는 **제품 허가사항 + KDIGO/전문가 가이드라인에 명시된 값** = 영양제보다 근거 등급 높음(A급).
- 단 eGFR cutoff는 가이드라인/국가별로 ±차이 있음(예: 메트포민 30 vs 45). **한국 식약처 허가사항과 대조 필요.**
- CrCl(Cockcroft-Gault) vs eGFR(CKD-EPI) 혼용 — 약물 라벨은 대개 CrCl 기준. 실무 환산 주의.

## 영선 검수 우선순위 (처방약)
1. **메트포민 한국 식약처 cutoff** (30 vs 45 — 국가별 다름)
2. **가바펜틴/프레가발린 단계별 용량** — 한국 허가사항과 일치?
3. **알로퓨리놀 + HLA-B*5801** — 한국인 권고 (한국 빈도 높음)
4. **콜키친 + CYP3A4 상호작용** — 약사 감사 포인트 톤
5. **니트로푸란토인 <30 금기** — 한국 처방 관행과 충돌 없나?
