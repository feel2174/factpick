## 🩺 한 줄 요약

**약사가 Cochrane 메타분석으로 검증한, 환자 맞춤 영양제·약 비교 사이트.**
정형외과 약·주사 맞아도 효과 며칠뿐인 무릎 통증 환자에게, "당신 상황에 진짜 맞는 옵션"을 1분 안에 추천.

🌐 **https://factpick.co.kr**

## 💭 왜 만들었나 — 운영자(영선)의 진짜 동기

> "관절에 좋다는 게 이렇게 많은데, 진짜 다 좋은 거 맞아?"
> "뭐가 제일 좋은 성분이지?"
> "어떤 게 진짜로 효과 있는 거야?"

약국에서 매일 듣는 질문이고, 사실 **약사인 영선도 답이 막막했던 질문**.
광고와 후기는 많은데, 진짜 데이터로 정리된 자료는 없었음.

그래서 **"나중에 내가 그 병에 걸리면 진짜로 뭘 먹어야 할까?"** 라는 기준으로 직접 만들기 시작.
Cochrane 메타분석·임상시험 데이터를 SMD(효과 크기) 수치로 비교하는 형태로 정리.

**마케팅 핵심 후킹 메시지로 활용 가능:**
- "약사도 답 못 했던 질문 — 진짜로 뭐가 좋은 영양제인가요?"
- "나중에 내가 그 병 걸렸을 때 진짜로 뭘 먹어야 하지?"
- "광고·후기 말고 데이터로 정리된 게 필요했어요"

이게 환자가 던지는 질문과 똑같음 → **공감 셀링의 출발점**.

## ✨ 핵심 차별화 (마케팅 셀링 포인트)

### 1️⃣ "주사 효과는 6개월이면 사라진다"는 정직한 데이터

대부분 영양제 사이트가 "이거 좋아요"라고만 외칠 때, FactPick은:
> **"관절강내 스테로이드 주사는 1-2주엔 효과 SMD -0.48, 6개월 지나면 -0.07로 위약 수준."**

이미 정형외과 주사·약 맞아도 효과 며칠뿐이셨던 환자가 **"내 얘기네"** 하고 반응할 후킹.

### 2️⃣ 데이터 출처가 **세계 표준 Cochrane Library**

- Cochrane 원문 PDF 5개에서 직접 추출
- forest plot에서 SMD/CI 직접 수치화
- 영양제 회사가 후원한 데이터가 아닌 **국제 표준**
- "글루코사민 Rotta SMD -1.11, 한국 시판 = 삼진제약 오스테민 캡슐" 같은 정확한 매칭

### 3️⃣ 약 + 영양제를 **같은 산점도**에 비교

기존 한국 사이트는 영양제 따로, 약 따로. FactPick은:
- 한 산점도에 영양제(초록) + 약(빨강)
- X축 근거 신뢰도 · Y축 효과 크기
- 우상단 = "효과 크고 근거 강함"

### 4️⃣ **1분 응답형 추천** (D2C quiz funnel)

```
키·몸무게 → 나이 → 동반질환 → 이전 시도 치료
→ 간기능 → 신기능 → 복용 약
→ ✨ Cochrane 데이터 기반 추천
```

- 단순 선택 화면은 클릭 즉시 다음 단계
- 결과: 1위 추천 카드 + 강한 구매 CTA + 차선 옵션

### 5️⃣ **"이미 해본 것"의 효과와 비교**

환자가 "물리치료·NSAID·관절 주사·일반 글루코사민" 등 시도해본 것 체크하면 결과 화면에:

```
지금까지 시도한 것 vs 추천
─────────────────────────
MSM              ▓░░░░░░░░░ 0.00
일반 글루코사민    ▓░░░░░░░░░ 0.05
관절 주사 (6개월)  ▓░░░░░░░░░ 0.07
NSAID 처방        ▓▓▓░░░░░░░ 0.22
─── 당신을 위한 추천 ───
오스테민 캡슐     ▓▓▓▓▓▓▓▓▓ 1.11 ✨
```

환자가 **숫자로 직접 차이를 봄** = 가장 강력한 설득 도구.

### 6️⃣ 환자 안전 = **약사 큐레이션 + 약학정보원 연동**

- 처방약은 affiliate X, "🏥 약국 처방 + 약학정보원 정보" 로 정직하게
- OTC도 "약사님께 ○○ 성분으로 문의하면 같은 성분 받을 수 있어요" 친절 안내
- 의료법 광고 규제 안전선 유지

## 🎯 사용자 흐름 (실제 경험)

### 1. 홈 진입
> "매번 주사 맞고, **맞을 때뿐**이셨나요?"
> [내게 맞는 옵션 찾기 →]

### 2. Quiz (1분, 7단계)
키·몸무게 → 나이대 → 동반질환 → **이미 시도한 치료** → 간기능 → 신기능 → 복용 약

### 3. 결과 화면
- 🥇 1위 추천 카드 (큰 폰트, 제품명, 제조사, SMD)
- 🛒 큰 구매 CTA (쿠팡 / iHerb / 약국 처방 / 약국 구입)
- 효과 강도 비교 막대 (이전 시도 vs 추천)
- 동반질환별 약사 메시지 카드
- 차선 옵션 3개
- "모든 데이터·산점도 더 보기" (펼침)

### 4. 더 보기 (전체 데이터 원하는 사람)
- 효능 vs 근거 산점도
- 41건 검증 + AI 추출 통합 비교표
- Cochrane 출처 직접 링크 (CD002946 등)

## 🛒 수익 모델

| 채널 | 방식 | 한도 |
|---|---|---|
| **쿠팡 파트너스** | 건강기능식품 ~3% 수수료 + 24시간 쿠키 효과 | 한국 영양제 시장 80%+ 커버 |
| **iHerb Rewards** | 추천 코드 적립 (JLN6158) | 직구 영양제 (쎄라큐민·강화제형 등) |
| **약학정보원 링크** | 수익 X, 신뢰도 자산 | 처방약·OTC 정보 |

- 같은 효과 구간 안에서는 affiliate 가능한 쪽 우선 정렬 (정직성 유지)
- 처방약·시술은 광고 없이 정보만 (의료법 안전선)

## 📸 시각 자료 (스크린샷 권장 위치)

홍보 콘텐츠에 쓸 사이트 스크린샷 추천 컷:

1. **인트로 화면** — "매번 주사 맞고, 맞을 때뿐이셨나요?" 후킹 카피
2. **Quiz 단계** (예: 동반질환 선택) — 깔끔한 카드 UI
3. **이전 치료 vs 추천** 비교 막대 차트 — 가장 강력한 시각 임팩트
4. **1위 추천 카드** + 쿠팡 CTA 버튼 — 행동 유도
5. **산점도** (모든 데이터 보기) — 전문성 자산

영선이 직접 사이트 한 번 돌며 캡처 (Mac에서 Cmd+Shift+4) → 홍보 콘텐츠에 사용.
